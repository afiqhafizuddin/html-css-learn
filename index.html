Posted by Laura Jones in Monday, June 21st 2027

All modern webistes and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the websites

In this post, let's ficus on HTML. We will learn what HTML is all about and why you should learn it

What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that web developers use to sructure and describe the content of a webpage (not a programming language.

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, videos, etc. Web browsers understand HTML and render HTNL code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs
